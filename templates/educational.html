Estrutura de programação em Python

a. Estruturas de Seleção (if, elif, else)

Permitem que o programa tome decisões com base em condições.

idade = 18 if idade >= 18: print("Você pode entrar no castelo!") else: print("Você é jovem demais, volte quando tiver 18.")
b. Estruturas de Repetição (for, while)

Executam um bloco de código várias vezes, de acordo com uma condição ou intervalo.

for i in range(5): print("Repetição número", i + 1)
c. Vetores e Matrizes (listas)

Listas armazenam múltiplos valores. Podem ser usadas como vetores (1D) ou matrizes (2D).

nomes = ["Ana", "Bruno", "Carlos"] print(nomes[0]) # Saída: Ana matriz = [[1, 2], [3, 4]] print(matriz[1][0]) # Saída: 3
d. Funções e Procedimentos (def)

Funções retornam valores; procedimentos apenas executam uma ação.

def somar(a, b): return a + b def mostrar_msg(): print("Operação concluída!") print(somar(3, 4)) # Saída: 7 mostrar_msg() # Saída: Operação concluída!
Extra: Tratamento de Exceções (try/except)

Permite lidar com erros de forma controlada, evitando que o programa quebre.

try: numero = int(input("Digite um número: ")) except ValueError: print("Você digitou algo que não é número.")
